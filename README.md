# VibeCoded LittleHome

Это форк [LittleHome](https://github.com/ktnk-dev/LittleHome) с изменениями, сделанными при помощи нейронки. Оригинальная идея сохранена: минимальная новая вкладка с часами, быстрым поиском и bang-командами. В этом форке основной фокус смещен на более удобный поиск, подсказки, настройки темы и быстрый запуск AI-промптов.

## Что изменено относительно оригинала

- Добавлено меню поисковых подсказок при вводе.
- Подсказки собираются из нескольких источников: поисковик, история браузера, закладки и открытые вкладки.
- Если подсказка указывает на уже открытую вкладку, `Enter` переключает на нее и закрывает текущую новую вкладку.
- Подсказки появляются сразу за счет локального fallback, а сетевые подсказки догружаются позже.
- Исправлены скачки меню подсказок: поле поиска остается по центру, список показывается под ним в фиксированной области.
- Переделан дизайн подсказок: общий стек без отдельных выбивающихся карточек, без тени, с аккуратной подсветкой совпавшей части запроса.
- Добавлены анимации появления и исчезновения подсказок через scale, position, blur и opacity.
- Подпись под поиском меняется в зависимости от выбранного результата: поиск Google, открытие истории, закладки, вкладки или URL.
- Исправлены bang-команды: `!ppx запрос` и другие валидные бенги снова открываются по `Enter`.
- Для бенгов в подсказках показываются и поисковой движок, и очищенный поисковой запрос.
- Добавлен AI-режим по `Space` на пустом поле: можно сразу ввести промпт для выбранной нейронки.
- В AI-режиме стрелками можно переключаться между ChatGPT, Perplexity, Gemini, Google AI Studio и Claude.
- Нейронка по умолчанию выбирается в настройках.
- Добавлена подсказка под часами: можно начать печатать для поиска или нажать пробел для AI-промпта.
- Добавлены настройки темы в нижнем левом углу.
- Светлая тема переименована в `int`, темная - в `sinya`.
- Восстановлен темный градиентный фон в стиле оригинального LittleHome, при этом светлая тема оставлена в текущем минимальном виде.
- Меню настроек получило анимацию появления и закрывается по клику вне меню.
- `Esc` очищает введенный текст и возвращает часы.
- Добавлены разрешения расширения для истории, закладок, вкладок и Google Suggest.
